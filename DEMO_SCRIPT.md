# 🎬 Сценарий демо-видео (2 минуты)

## Подготовка
- Открой OBS Studio или любую программу для записи экрана
- Включи запись голоса (микрофон)
- Открой сайт http://<VM_IP>:8083 в браузере

## Сценарий по секундам

### 0:00 - 0:15 — Вступление
> "Это Holiday Agent — AI-ассистент для управления праздниками. Задавайте вопросы на естественном языке, и агент мгновенно ответит."

### 0:15 - 0:35 — Dashboard
> "При открытии сайта загружается дашборд. Здесь статистика, обратный отсчёт до следующего праздника, графики по категориям и месяцам."
- Покажи countdown
- Прокрути до "Upcoming Holidays"
- Наведи на даты

### 0:35 - 0:55 — Calendar
> "Переключаемся на календарь. Праздники подсвечены цветами по категориям. Кликните — увидите описание."
- Кликни на вкладку Calendar
- Кликни на любой праздник → покажи описание внизу
- Прокрути месяцы ◀ ▶

### 0:55 - 1:25 — Chat
> "Теперь чат. Спрашиваю на естественном языке."
- Кликни Chat
- Напиши: "Show me national holidays" → покажи ответ
- Нажми **+ Add** → покажи 3-шаговый процесс (name → date → category)
- Или напиши: "What is the nearest holiday?"

### 1:25 - 1:45 — Notifications & Delete
> "Колокольчик включает уведомления. А красная кнопка ✕ удаляет праздник."
- Нажми 🔔 → покажи toast-уведомление
- На дашборде нажми ✕ рядом с любым праздником → подтверди удаление

### 1:45 - 2:00 — Export & Заключение
> "Кнопка экспорта скачивает .ics файл с напоминаниями. Его можно импортировать в Google или Apple Calendar."
- Нажми 📥 → покажи скачанный файл
- "Holiday Agent работает на VM по адресу http://VM_IP:8083. Код открыт на GitHub."

## Советы
- Говори чётко, без пауз
- Не торопись — 2 минуты достаточно
- Перед записью отрепетируй 1-2 раза
- Убедись, что Qwen API healthy (`curl localhost:42005/health`)
- Очисти test holidays перед записью
